Document Translator

This document translator takes PDF, .doc, .docx or HTML document and uses IBM's Watson Document Conversion service to fetch the plain text representation of it. Then it takes the plain text and translates (only from English to Portuguese) it by using IBM's Watson Language Translator service. After doing that, it outputs the result of both services on the HTML.
